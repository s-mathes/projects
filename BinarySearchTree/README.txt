This program implements a binary search tree that the user can add numbers to, delete numbers
from, and search for numbers. The program can also print the binary search tree in post-order,
pre-order, in-order, and breadth first, as well as its height.

To compile, use the command 'make' to compile the executable 'main'.

Commands:
'X' to print binary search tree info
'Q' to quit program
's #' to search for a number
'#' to add a number to the tree
'?' for instructions
